# Market Regime Analysis – Trading Behavior vs Market Sentiment

## Project Overview
This project analyzes how trading behavior (profitability, risk exposure, volume, and emotional intensity)
varies across different market sentiment regimes derived from the Fear & Greed Index.

The objective is to identify hidden behavioral patterns and market regimes that can inform smarter,
data-driven trading strategies.

## Structure
- `notebook_1.ipynb` – Main Google Colab notebook with full analysis
- `csv_files/` – Raw and processed datasets
- `outputs/` – Visual outputs and charts
- `ds_report.pdf` – Final summarized report with insights and conclusions

## Key Highlights
- Correct handling of Unix timestamps and date alignment
- Regime-based behavioral analysis
- Risk–reward, volume, and psychology-driven insights
- Market regime dashboard for decision-making

## Tools Used
- Python (Pandas, NumPy)
- Matplotlib, Seaborn
- Google Colab
